Récupérer les valeurs des checkbox avec PHP

récupération des valeurs des checkbox cochées dans un formulaire

Dans ce tutoriel PHP, nous allons voir comment faire pour récupérer les valeurs des checkbox qui sont cochées dans un formulaire html. Pour ce faire, nous allons utiliser deux notions de PHP: les tableaux et la structure foreach.

Première étape: créer un formulaire html avec des checkbox

Pour pouvoir récupérer les valeurs des checkbox cochées par vos visiteurs, vous devez nommer les checkbox du formulaire avec des crochets [], c'est grâce à cette notation que PHP peut comprendre que plusieurs valeurs peuvent potentiellement être envoyées dans un groupe de checkbox.

<form action="index.php" method="post">
   <input type="checkbox" name="prenom[]" value="adriana" />Adriana<br>
   <input type="checkbox" name="prenom[]" value="alessandra" />Alessandra<br>
   <input type="checkbox" name="prenom[]" value="candice" />Candice<br>
   <input type="checkbox" name="prenom[]" value="lili" />Lili<br>
   <input type="submit" value="Envoyer" />
</form>

Dans l'exemple ci-dessous, vous pouvez voir que la valeur donnée à l'attribut "name" des checkbox est "prenom[]" et pas "prenom".

Deuxième étape: récupérer la valeur des checkbox

La notation par crochets placée dans l'attribut "name" des checkbox va nous permettre de récupérer leur valeur très facilement en utilisant la structure foreach(). En effet, foreach() va lire une par une les lignes du tableau dans lequel ont été insérées les valeurs cochées par l'utilisateur une fois qu'il a cliqué sur le bouton "envoyer" du formulaire.

// Le tableau $_POST['prenom'] contient les valeurs des checkbox cochées
foreach($_POST['prenom'] as $valeur)
{
   echo "La checkbox $valeur a été cochée<br>";
}

/*
Si l'utilisateur coche Adriana et Candice, 
cet exemple va afficher:
La checkbox Adriana a été cochée
La checkbox Candice a été cochée
*/

Dans cet exemple, toutes les valeurs cochées se trouvent dans le tableau $_POST['prenom'] , "prenom" étant le nom que nous avons donné aux checkbox dans le formulaire. Ensuite, la structure foreach() lit une par une les lignes du tableau et la valeur de chaque ligne est renvoyée dans la variable $valeur.

La traduction en français de cet exemple est la suivante: pour chaque ligne présente dans le tableau prenom, afficher sa valeur.

Si aucune checkbox n'a été cochée, le tableau PHP sera vide et aucune valeur ne sera retournée dans la boucle foreach().

if(!$_POST['prenom']){
   echo "Aucune checkbox n'a été cochée";
}
